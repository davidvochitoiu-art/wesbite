# Improved Programming Website

## How to use
1. Replace your current CSS folder with this CSS folder.
2. Replace your current Html folder with this Html folder if needed.
3. Replace your current Image folder with this Image folder.
4. Open `Html/home.html` in your browser.

## What changed
- Bigger readable fonts across all pages.
- Better spacing and line height.
- Removed risky fixed `min-width: 2000px` style problems.
- Improved mobile/tablet responsiveness.
- Stronger dark overlays for readability.
- Better hover animations and glowing cyber style.
